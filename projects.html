More Projects

Deck Manager for Clash Royale

A small Android app developed in Java used to store card decks for the mobile game Clash Royale. Can no longer maintain it as the Google PlayStore reviewers don't let me update the app because they think I don't have permission to use the Clash Royale image assets. Unfortunately they don't care that it's fair use as per Supercell's fan content policy. For now it's still available in the PlayStore here, because the reviewer for the initial publishing either didn't care or realized that it's fair use.

Grindy Alarm

Grindy Alarm is an Android app that helps people for whom it is a "grind" to wake up in the morning. It stops one from falling back asleep after turning off the alarm. I developed this app natively in Java. These days I no longer actively maintain it as Android updates and device manufacturer's "battery optimizations" have made it practically impossible to provide a satisfying experience for all users.

Proximity-Scroll

A proof of concept that shows that it's possible to scroll on websites using the smartphone's proximity sensor instead of touching the screen. Developed for the Mobile Computing & Internet of Things lecture at KIT You can try it yourself here.
